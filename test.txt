2.	~ - домашняя директория пользователя
	. - текущая директория

3. rm - команда удаления файлов и директорий

5. cp - команда копирования файлов и директорий

7. mv старое_название новое_название
8. mv путь_до_файла новый_путь
9. Флаг -R используется в командах, чтобы применить их рекурсивно ко всем файлам и поддиректориям
10. Флаг -r используется, чтобы рекурсивно обработать поддиректории

12. git init - команда для инициализации локального репозитория
13. git checkout -b "название"
14. git checkout "название"
15. Процесс объединения изменений из одной ветки в другую
